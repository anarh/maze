# maze
Maze for fun

## Requirements
Node v4+, npm v2+

## Run Command Line
1. CLONE repo
2. RUN `node index.js` (uses default example maze file)
3. or RUN `node index.js maze.txt` (for your own custom file)

## Run in Browser
1. CLONE repo
2. RUN `npm install`
3. OPEN `/example/index.html` in your browser
